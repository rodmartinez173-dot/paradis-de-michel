# Le Paradis de Michel

Site humoristique et affectueux en hommage à tous les Michel de France.

- **Accueil** : galerie du Michel standard (apéro, pétanque, barbecue, sieste...)
- **Les Michel célèbres** : le Panthéon des dix plus grands Michel, avec bios et anecdotes
- **Les chansons** : le juke-box des chansons qui portent le prénom, vidéos incluses

Site statique (HTML/CSS/JS pur), design créé avec Claude Design et implémenté avec Claude Code.

## Consultation locale

Double-cliquer sur `lancer-le-site.bat` (Windows, Python requis) : les vidéos YouTube ne s'affichent pas en ouvrant `index.html` directement en `file://`.

*Aucun Michel n'a été maltraité pendant la fabrication de ce site.*
